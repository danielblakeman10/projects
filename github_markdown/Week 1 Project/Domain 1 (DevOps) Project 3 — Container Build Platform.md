# Project 3 — Container Build Platform

## Project

Create a centralized container build platform.

Common Things

1. CI/CD with GitHub Actions

2. Infrastructure as Code with either Terraform/Cloud Formation/CDK

3. Security Tool – Use security tools with either GitHub Secrets or environment variables or AWS Secrets Manager. Use an IAM role and put that in an environmental variable.

4. Build using EC2 and put it in the container.

## Features

- Docker Build

- Multi-arch builds

- Vulnerability scanning

- ECR publishing

- Automated tagging strategy
